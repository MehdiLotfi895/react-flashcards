# React Flashcards App üìù

A simple and interactive React application designed to help users learn core React concepts through flip-cards.

## üöÄ Features
* **Interactive UI:** Click on a card to flip between the question and the answer.
* **State Management:** Uses the `useState` hook to track which card is currently selected.
* **Dynamic Rendering:** Maps through a list of educational content to generate cards dynamically.
* **Toggle Logic:** Clicking a card reveals the answer, and clicking it again (or clicking another) resets the view.
* **Custom Styling:** Features a clean layout with dedicated CSS for card dimensions, colors, and alignment.

## üß† Topics Covered
The app includes flashcards for the following essential concepts:
* **JavaScript:** The base language React is built on.
* **Components:** The fundamental building blocks of React apps.
* **JSX:** The syntax used to describe UI in React.
* **Props:** Passing data from parent to child components.
* **useState:** Giving components memory and managing state.
* **Controlled Elements:** Synchronizing input elements with state.

* **React.js:** Core library used for the UI and logic.
* **CSS3:** For layout (Flexbox) and card design.
* **JavaScript (ES6):** Used for data structures and logic handling.
